The Tech Powering InvestWise

Our platform is engineered with a cutting-edge technology stack and sophisticated machine learning models to deliver robust, scalable, and accurate financial intelligence.

Scalable Architecture

The core InvestWise platform (not this website) utilizes Django, TensorFlow, React (for platform UI), PostgreSQL, Redis, Docker, and Celery for a resilient and high-performance system.

Stack: Django, TensorFlow, React, PostgreSQL

Advanced ML Models

Employing LSTMs for GDP forecasting, Random Forests/MLPs for investment recommendations, and NLP for sentiment analysis of sector-specific news and reports.

Models: LSTM, Random Forest, NLP

Comprehensive Data Sources

Insights derived from KNBS, World Bank, CBK, KRA, NSE, global financial news, alternative data, and user-inputted parameters for simulations.

Data: KNBS, World Bank, CBK, NSE, News APIs
