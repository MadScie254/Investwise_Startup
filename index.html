The Engine Behind InvestWise

Our platform is built on a robust, scalable architecture and state-of-the-art machine learning models to deliver unparalleled analytical capabilities.

System Architecture

The InvestWise platform utilizes a modern stack: Django for the backend API, TensorFlow for ML model serving, React for the dynamic platform UI (not this website), PostgreSQL for data storage, Redis for caching, Docker for containerization, and Celery for asynchronous tasks.

Tech: Django, TensorFlow, React, PostgreSQL, Docker

Machine Learning Models

We employ a suite of ML models: LSTMs for time-series GDP forecasting, Random Forests/MLPs for investment recommendations, and advanced NLP techniques for sector health analysis based on sentiment and news.

Models: LSTM, Random Forest, MLP, NLP

Diverse Data Sources

Our insights are fueled by a rich variety of data from trusted sources including KNBS, World Bank, Central Bank of Kenya (CBK), KRA, Nairobi Securities Exchange (NSE), global financial news, and user-inputted data for personalized simulations.

Sources: KNBS, World Bank, CBK, NSE
